# Issue #006: 酒杯颜色红粉橙黄区分度不足

## 状态：已修复 ✅

## Commit: `db2f90d`

## 问题描述

当前 8 种酒杯颜色中，部分颜色在色相环上过于接近，导致快速游玩时容易混淆：

1. **红色 (#FF4757, H≈355°) 与 粉色 (#FF6B81, H≈350°)** - 色相仅相差 5°，都在红色系，难以快速区分
2. **橙色 (#FFA502, H≈39°) 与 黄色 (#FFDD59, H≈48°)** - 色相仅相差 9°，都在暖色调，区分度不够

## 色相分布分析

修复前的色相分布：
- 红色: 355°
- 粉色: 350° ← 与红色过近
- 橙色: 39°
- 黄色: 48° ← 与橙色过近
- 绿色: 145°
- 青色: 181°
- 蓝色: 210°
- 紫色: 270°

## 修复方案

### 第一步：调整颜色

调整粉色和黄色，使色相分布更均匀：

1. **粉色** → **洋红色 (Magenta)** `#FF1493`
   - 色相约 328°，与红色拉开 27° 距离
   - 更偏向紫红色，与纯红色明显区分

2. **黄色** → **金黄色** `#FFD700`
   - 色相约 51°，与橙色保持合理间距
   - 更纯正的金黄色，与橙色区分更明显

### 第二步：添加形状区分

为每种酒杯类型设计独特的形状，即使颜色相近也能快速识别：

| 类型 | 颜色 | 形状 |
|------|------|------|
| 0 | 红色 #FF4757 | 圆形 (Circle) |
| 1 | 绿色 #2ED573 | 方形 (Square) |
| 2 | 蓝色 #1E90FF | 三角形 (Triangle) |
| 3 | 橙色 #FFA502 | 菱形 (Diamond) |
| 4 | 紫色 #A855F7 | 六边形 (Hexagon) |
| 5 | 金黄色 #FFD700 | 星形 (Star) |
| 6 | 洋红色 #FF1493 | 心形 (Heart) |
| 7 | 青色 #00D2D3 | 五边形 (Pentagon) |

通过颜色+形状的双重区分，大幅提升了酒杯类型的辨识度。
