# Issue #001: 酒杯颜色区分度不足

## 状态：已修复 ✅

## 问题描述

游玩过程中发现不同类型酒杯的颜色过于相近，容易导致玩家误判。

具体问题颜色对：
- teal `#4ECDC4` 与 mint `#98D8C8` 色相接近
- blue `#45B7D1` 与 teal `#4ECDC4` 色相接近
- green `#96CEB4` 与 mint `#98D8C8` 色相接近

## 修复方案

重新设计 8 色调色板，确保相邻颜色在色相环上均匀分布（间隔约 45°），同时保持高饱和度和明度以便在深色背景上清晰辨认。

## 修改文件

- `src/renderer/canvas-renderer.ts` — `GLASS_COLORS` 数组
