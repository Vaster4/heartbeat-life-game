# Feature #005: 难度梯度与障碍石

## 状态：已完成 ✅

## Commit: `71dbd0c` (dev)

## 版本：0.2.0

## 需求描述

### 1. 难度梯度 - 酒杯类型渐进解锁
- 游戏开始时只有 3 种酒杯类型
- 每次目标酒杯刷新时，解锁 1 种新酒杯类型（直到达到上限 8 种）

### 2. 目标酒杯刷新进度调整
- 初始刷新阈值：4（原来是 10）
- 每次刷新后阈值增长 2（即 4, 6, 8, 10, ...）

### 3. 目标酒杯刷新进度条 UI
- 美观的能量进度条样式
- 显示当前进度 / 总需求

### 4. 障碍石机制
- 游戏开始时随机放置 4 个障碍石
- 盘子不能放在障碍石上
- 每个障碍石有 1~3 层封印
- 相邻位置（上下左右）完成满6消除 → 破除 1 层封印
- 封印完全破除 → 获得 初始封印层数 × 3 分
- 破除障碍石获得当前目标酒杯刷新阈值 50% 的进度，溢出累计到下轮
- 目标酒杯刷新后，在最多 2 个空格中随机刷出新障碍石（全场上限 6 个）

### 5. 障碍石 UI
- 1 层封印：石质外观
- 2 层封印：铁质外观
- 3 层封印：钻石质外观
- 用破损程度表示剩余封印层数
