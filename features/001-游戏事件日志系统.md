# Feature #001: 游戏事件日志系统

## 状态：已完成 ✅

## 需求描述

为游戏添加完整的事件日志系统，记录所有游戏移动和合并事件，方便调试和问题排查。

## 实现内容

- 新增 `GameLogger` 类，支持 INFO / DEBUG / WARN 三个级别
- 记录以下事件类别：
  - `GAME` — 游戏开始、游戏结束
  - `ROUND` — 新轮次开始、发放盘子详情
  - `SELECT` — 选中临时区盘子
  - `PLACE` — 放置盘子到棋盘（含坐标、时间戳、酒杯内容）
  - `MERGE` — 合并事件（源→目标、类型、数量）
  - `ELIM` — 消除事件（满盘/空盘、combo、得分）
  - `SCORE` — 每次操作的得分汇总
  - `BONUS` — 轮次奖励触发
  - `TARGET` — 目标酒杯刷新
- 日志同时输出到 console
- 支持导出为 `.txt` 文件下载（浏览器环境）
- 按 `L` 键可快速下载日志文件
- 控制台可通过 `window.gameLogger` 访问日志对象

## 修改文件

- `src/core/logger.ts` — 新增 GameLogger 类
- `src/core/game-engine.ts` — 集成日志到所有关键事件
- `src/main.ts` — 绑定快捷键和 window 暴露
