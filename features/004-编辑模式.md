# Feature #004: 编辑模式（Edit Mode）

## 概述
提供一个编辑模式，允许用户手动编辑棋盘和临时区的盘子内容，然后以编辑后的状态开始游戏，用于测试和调试特定场景。

## 操作方式

### 进入/退出
- 按 `E` 键进入编辑模式
- 按 `E` 或 `Enter` 退出编辑模式，加载编辑后的状态开始游戏

### 编辑操作
- 点击棋盘格子或临时区槽位 → 选中（金色虚线高亮），空位自动创建空盘子
- 数字键 `0-7` → 向选中盘子添加对应类型的酒杯（最多6个）
- `Backspace` → 删除选中盘子的最后一个酒杯（酒杯清空后盘子自动移除）
- `Delete` → 清空整个选中格子

### 视觉反馈
- 顶部红色横幅显示"编辑模式"及操作提示
- 棋盘周围红色虚线边框
- 选中格子金色虚线高亮

## 技术实现
- `src/main.ts`: EditState 状态管理、键盘/点击事件处理
- `src/core/game-engine.ts`: `loadEditedState()` 方法加载编辑后的棋盘和临时区
- `src/renderer/canvas-renderer.ts`: `editMode`/`editSelection` 属性、`drawEditModeOverlay()` 绘制

## 状态重置
退出编辑模式时，得分、轮次、combo 等状态全部重置为初始值。
